# TravelTrek

**TravelTrek** is a full-stack travel guide web application that helps users explore destinations by offering curated information on attractions, accommodations, and dining options. Designed to enhance the travel experience, the platform uses embedded Google Maps (via iframe) for intuitive location reference.

## Features

- Discover popular tourist destinations
- View recommended stays and local dining options
- Embedded Google Maps for quick location context
- Clean and responsive user interface
- Backend support to manage and store travel data

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Mapping:** Google Maps (via iframe)

## Future Enhancements

- User authentication and itinerary planner
- Admin dashboard for content management
- Integration with third-party travel or booking APIs
- User reviews, ratings, and feedback system
